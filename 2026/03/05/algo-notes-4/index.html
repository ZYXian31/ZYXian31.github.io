今日总结

1. 避免递归中构造新列表

append 优于 list +

2. 注意 Python 作用域

函数中赋值默认局部变量，需要修改外层变量时使用 nonlocal

3. 避免重复计算

能一次 DFS 算完的，不要拆成多个递归

4. 避免递归切片

切片隐藏 O(n)，优先用下标区间

5. 回溯只恢复状态
